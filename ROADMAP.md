# Roadmap

## 0.1.0

- Task package schema and examples.
- CLI commands:
  - `init`
  - `check`
  - `validate-final-delivery`
  - `validate-receipt`
- Unit tests with the Python standard library.
- GitHub Actions test workflow.

## 0.2.0

- JSON schema validation for all public schemas.
- More examples:
  - release task
  - cron/checkback task
  - human approval gate
- Machine-readable validator reports.

## 0.3.0

- GitHub issue and PR maintenance examples.
- Optional GitHub Actions composite action.
- Recovery playbook templates.

## Later

- Provider-neutral delayed job tracking examples.
- Multi-agent handoff examples.
- Independent evaluator workflow examples.
